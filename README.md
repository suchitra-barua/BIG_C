# BIG_C
GroceryHub is a full-stack web-based grocery shopping platform that allows users to browse, purchase, and manage grocery items online. The system is designed to simulate a real-world e-commerce environment with features for customers and administrators.
Frontend: HTML, CSS, Tailwind CSS
Backend: PHP
Database: MySQL (included SQL file: grocerydb.sql)
Architecture: MVC-style structure (Controller & Viewer separation)
